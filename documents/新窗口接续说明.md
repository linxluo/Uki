## 本窗口（2026-05-23）完成内容

### 第 15 课：插件系统
- `uki/plugin_manager.py`：插件管理器，支持发现/加载/生命周期/启用禁用/卸载
- `plugins/time_utils/`：示例插件（2 工具 + 1 命令）
- 插件通过 `uki_plugin.json` 声明身份和依赖（自动 pip install）
- `/plugin` 命令：查看状态、安装 zip、启用/禁用
- `/createMyPlugin` 命令：LLM 自动生成插件代码 → 打包 zip → 自动安装
- 插件工具优先级：插件 > MCP > 内置

### S3：执行命令与沙箱（门禁模式）
- `uki/tools.py`：新增 `execute_bash` 工具，`shell=True` 自适应 OS
- 安全：11 个危险命令模式自动拦截（含 Linux + Windows）、10-300s 超时、8000 字符输出截断
- 权限：加入 `WRITE_TOOLS`，default 模式执行前需确认
- 工具描述动态注入 OS 信息（Windows 用 dir/del，Linux 用 ls/rm）

### 前端 UI 全面升级
- 回复 Markdown 渲染：逐行状态机解析，支持标题/加粗/列表/代码块/引用/分隔线
- 思考过程可折叠：默认折叠，点击 ▸ 展开查看 💭🔧 中间步骤
- 三个跳动加载点：Agent 推理时显示●●●动画
- 附件栏：拖入文件上传，自动提示 LLM 处理
- 命令自动补全：输入 / 弹出下拉菜单，模糊过滤，↑↓ 选择
- 设置页新增「文件缓存」tab：查看/删除已上传文件
- 设置页新增「插件」tab：拖入 zip 安装 + 已安装列表 + 开关
- 配置保存即时生效，无需重启
- 命名标签：用户气泡显示「你」，回复显示「Uki」

### 课程更新
- 补充课程新增 S4（插件自动建议）：依赖第 16 课记忆系统
- 补充课程新增 S5（执行沙箱真隔离）：进程/文件系统/网络/资源限制

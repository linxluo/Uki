## 项目当前状态

```
UkiAgent/
├── UKI.md              # 项目规则文件（Uki 每次启动自动读取）
├── README.md           # 项目说明 + 开发日志
├── requirements.txt    # Python 依赖（openai, python-dotenv）
├── .env.example        # API 配置模板
├── .gitignore
├── main.py             # 入口文件
└── uki/
    ├── __init__.py
    ├── config.py       # 配置管理
    ├── agent.py        # 核心 Agent 类（代理循环 + 上下文管理）
    ├── tools.py        # 工具集（list_files, read_file, write_file, search_code）
    └── commands.py     # 斜杠命令系统（/help /tools /config /model /clear /compact /context）
```

**Uki 当前能力**：
- 代理循环（思考→行动→观察，最多10轮）
- 文件操作（列出目录、读取、写入）
- 代码搜索（文件名 glob + 内容正则）
- 斜杠命令（7个本地命令，不消耗 token）
- 规则文件系统（UKI.md 自动注入 system prompt）
- 上下文管理（token 估算 + 自动裁剪）

**GitHub**：https://github.com/linxluo/Uki.git

## 补充课程（Claude Code 文档外的重要内容）

> 这些是学习过程中发现的重要主题，Claude Code 没有单独成章但在工程实践中必不可少。

| 编号 | 主题 | 说明 | 状态 |
|------|------|------|------|
| S1 | Prompt 自动增强 | Agent 在发给 LLM 前自动处理上下文，包含以下子项： | ⬜ |
| S1.1 | 上下文自动增强 | 自动附上相关文件、注入 Git 状态、项目结构信息 | ⬜ |
| S1.2 | 翻页主动清理 | LLM 翻页时标记跨页关联，精准保留被引用的旧页，清理无用页 | ⬜ |
| S1.3 | 翻页轮数限制 | 单次任务中限制工具翻页的最大轮数，防止无限膨胀 | ⬜ |
| S1.4 | 代码智能接入 | 通过跳转定义、类型检查等减少逐行翻文件的需求 | ⬜ |
| S1.5 | 超大用户输入处理 | 自动分段 + 索引，不是全量塞入；超限时弹确认提示 | ⬜ |
| S1.6 | System Prompt 长度管控 | UKI.md 和 Skill 内容设长度上限，超出截断并提示 | ⬜ |
| S2 | 网络搜索集成 | 为 Uki 接入联网搜索能力（MCP 演示案例） | ⬜ |
| S3 | 执行命令与沙箱 | 给 Uki 执行命令的能力（排在权限配置之后） | ✅ |
| S4 | 插件自动建议（信号二） | Uki 观察到反复生成同一脚本，主动建议做成插件。依赖第 16 课记忆系统。 | ⬜ |
| S5 | 执行沙箱（真隔离） | 进程级隔离（Windows Job Object / cgroup）、文件系统限制、网络限制、资源上限。区别于 S3 的门禁模式。 | ⬜ |
